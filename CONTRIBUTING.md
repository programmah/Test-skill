# Contributing

Thank you for contributing to `Test-skill`.

## Development Flow

1. Create a branch from `main`.
2. Make focused changes with clear commits.
3. Update documentation when behavior changes.
4. Open a pull request with a summary and validation notes.

## Validation

Before opening a pull request, run any project-specific tests and include the
commands and results in the pull request description.
