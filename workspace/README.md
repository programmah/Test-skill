# Workspace

This folder contains the bootcamp material for `Test-skill`.

- `start_here.ipynb`: overview, prerequisites, table of contents, duration, audience, and licensing notes.
- `jupyter_notebook/`: hands-on Jupyter notebook labs.
- `jupyter_notebook/images/`: images referenced by notebook labs.
- `images/`: shared images used across notebooks and documentation.
- `source_code/`: Python scripts and supporting source files.
- `data/`: datasets or dataset download notes when required.
- `model/`: models or model download notes when required.
- `challenge/`: challenge-lab assets and instructions.
